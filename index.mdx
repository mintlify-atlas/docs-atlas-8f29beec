<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-lg text-gray-600 dark:text-gray-400 mb-8">Get up and running with TechHelpDesk API in minutes</p>

  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install the required packages:
      
      ```bash
      git clone https://github.com/SantiagoPortillo1/Prueba-de-Desempe-o-M-dulo-Node.js-con-NestJS-y-TypeORM.git
      cd Prueba-de-Desempe-o-M-dulo-Node.js-con-NestJS-y-TypeORM
      npm install
      ```
      
      <Note>Requires Node.js v16 or higher and PostgreSQL v12 or higher</Note>
    </Step>
    
    <Step title="Configure your environment">
      Set up your database and environment variables:
      
      ```bash
      cp .env.example .env
      # Edit .env with your database credentials
      npm run migration:run
      ```
      
      See the [installation guide](/installation) for detailed configuration options.
    </Step>
    
    <Step title="Start the server">
      Launch the API server in development mode:
      
      ```bash
      npm run start:dev
      ```
      
      The API will be available at `http://localhost:3000` with Swagger documentation at `http://localhost:3000/docs`
    </Step>
    
    <Step title="Make your first API call">
      Authenticate and create your first ticket:
      
      ```bash
      # Login
      curl -X POST http://localhost:3000/auth/login \
        -H "Content-Type: application/json" \
        -d '{"email":"admin@example.com","password":"password123"}'
      
      # Create a ticket (use the token from login response)
      curl -X POST http://localhost:3000/tickets \
        -H "Authorization: Bearer YOUR_TOKEN" \
        -H "Content-Type: application/json" \
        -d '{"title":"Server issue","description":"Server not responding","clientId":"...","categoryId":"..."}'
      ```
    </Step>
  </Steps>
</div>
